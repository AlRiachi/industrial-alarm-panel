# Security Policy

## Supported Versions

Security fixes target the latest released version of Industrial Alarm Panel.

## Reporting A Vulnerability

Do not open a public GitHub issue for a security vulnerability.

Use GitHub private vulnerability reporting if it is available on the repository. If private reporting is unavailable, open a minimal public issue that says you need a private contact path, without exploit details, secrets, logs, or screenshots.

Please include:

- affected version
- Home Assistant version
- impact summary
- reproduction steps
- whether the issue requires authenticated Home Assistant access

## Scope

This integration runs inside Home Assistant and follows Home Assistant's authentication and authorization model. Reports about exposed Home Assistant instances, weak passwords, or unrelated add-ons should be directed to the affected project.
